*Aplicação para eleições diárias de restaurantes, feito em Nodejs e Angularjs. O Front-End pode ser desacoplado e hospedado em um local diferente do Back-end. Aplicação REST.*
## Get Started
```{r, engine='bash'}
npm install
bower install
sails lift 
```
